Le projet D.E.L.T.A est une intelligence artificielle inspirée de l'IA JARVIS d'Iron Man. J'ai créé cet assistant pour réaliser l'un de mes rêves d'enfance : posséder mon propre Jarvis. Ce projet a été concrétisé grâce à divers outils, notamment des vidéos explicatives sur YouTube et des documentations techniques qui m'ont permis de simplifier le code et d'optimiser le développement.

D.E.L.T.A repose sur une base d'IA existante via une clé API, ce qui m'a permis de me concentrer sur les fonctionnalités les plus importantes. Au-delà d'un chatbot classique, Delta a accès à la webcam de l'appareil pour décrire avec précision son environnement, ce qui rend ses réponses plus cohérentes et contextuelles. Il est capable d'ouvrir des liens dans le navigateur web et dispose d'une interface vocale permettant une interaction naturelle.

Doté d'une mémoire pour retenir les informations essentielles, D.E.L.T.A évolue constamment. À l'avenir, il disposera d'un contrôle total sur l'appareil hôte, sera accessible sur mobiles (smartphones et tablettes) et pourra gérer les contacts, passer des appels, envoyer des messages ou encore administrer entièrement la boîte mail de l'utilisateur.
