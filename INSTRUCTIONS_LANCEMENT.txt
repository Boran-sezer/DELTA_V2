🔧 EN CAS DE PROBLÈME:

Problème: Le backend ne démarre pas
Solution: 
  cd backend
  pip install -r ../requirements.txt
  python server.py

Problème: Interface ne s'ouvre pas
Solution: Vérifier que les ports 5173 et 8000 sont libres

Problème: Erreur de clé API
Solution: Vérifier le fichier .env avec vos clés Gemini
